# FleetMaster Design Analysis

## Brand Identity
- **Name:** FleetMaster
- **Tagline:** Premium Car Rentals
- **Style:** Clean, professional dashboard interface

## Color Scheme
- **Primary Blue:** Used for sidebar and active navigation (#2563EB or similar)
- **Light Background:** Off-white/light gray (#F9FAFB)
- **Card Background:** White with subtle shadows
- **Accent Colors:**
  - Green (Available status)
  - Blue (Rented status)
  - Red (Maintenance status)
  - Yellow (Warnings/Alerts)

## Layout Structure
- **Sidebar Navigation:** Fixed left sidebar with logo, navigation items, and sign out at bottom
- **Main Content Area:** Dashboard with metric cards and charts
- **Grid System:** 4-column metric cards, 2-column chart layout

## Key Components

### Metric Cards
1. Total Fleet (with car icon, blue background)
2. Revenue Est. (with dollar icon, green background)
3. In Maintenance (with wrench icon, red background)
4. Expiring Docs (with warning icon, yellow background)

### Charts
1. **Fleet Status:** Donut/semi-circle chart showing Available, Rented, Maintenance
2. **Fleet Composition:** Bar chart showing distribution by category (Economy, Luxury, SUV)

### Navigation
- Dashboard (home icon)
- Fleet (car icon)
- Sign Out (logout icon at bottom)

## Typography
- Clean sans-serif font (likely Inter or similar)
- Bold headings for section titles
- Regular weight for body text
- Numeric values emphasized with larger font size

## Interactive Elements
- Hover states on navigation items
- Card shadows for depth
- Responsive charts
- Sign out button
